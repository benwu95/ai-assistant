# Taiwan 用語對照（必須遵循）

> 同一份輸出內用詞必須統一。

| 中國用語（避免） | 台灣用語（採用） | 英文對應 |
|---------|--------|------|
| 異步 | 非同步 | async |
| 同步（情境用法）| 同步 | sync（仍可用「同步」，台灣亦通用）|
| 模板 | 範本 | template |
| 流水線 | 產線 / 管線 | pipeline |
| 迭代 | 疊代 | iteration |
| 範式轉移 | 典範轉移 | paradigm shift |
| 審計 | 稽核 | audit |
| 函數 | 函式 | function |
| 調用 | 呼叫 | call |
| 內存 | 記憶體 | memory |
| 概率分布 | 機率分佈 | probability distribution |
| 復制粘貼 | 複製貼上 | copy-paste |
| 循環 | 迴圈 | loop |
| 組件 | 元件 | component |
| 編碼（手動）| 手刻程式 / 寫程式 | coding（「編碼」在台灣指 encoding） |
| 閉環 / 閉環迴圈 | 封閉迴圈 | closed loop |
| 開發力 | 開發能力 | dev capability |
| 用戶 | 使用者 | user |
| 對象（物件意義）| 物件 | object |
| 線程 | 執行緒 | thread |
| 進程（process）| 行程 / 程序 | process |
| 默認 | 預設 | default |
| 數據 | 資料 | data |
| 字符串 | 字串 | string |
| 接口 | 介面 | interface |
| 隊列 | 佇列 | queue |
| 緩存 | 快取 | cache |
| 消息 | 訊息 | message |
| 文件（指 file）| 檔案 | file |
| 標識 | 識別 | identifier |
| 升級 | 升級 | upgrade（同）|
| 退化 / 降級 | 降級 | downgrade |
| 提交（git）| 提交 / commit | commit |

## 避免歧義斷句

造詞時確認不會被中文連讀誤解，例如「能量化」（讀成「能量／化」）→ 改用「可量化」。優先用「可X」而非「能X」。

## 中英文混排空格

英文單字 / 識別字 / code 前後必須加半形空格（行內 `` `code` `` 反引號外亦同）。
