## 避免歧義斷句

造詞時確認不會被中文連讀誤解，例如「能量化」（讀成「能量／化」）→ 改用「可量化」。優先用「可X」而非「能X」。

## 中英文混排空格

英文單字 / 識別字 / code 前後必須加半形空格（行內 `` `code` `` 反引號外亦同）。
